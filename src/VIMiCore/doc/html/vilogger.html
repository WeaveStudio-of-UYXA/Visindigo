Detailed Description

VILogger定义了Visindigo的日志记录器，所有Visindigo的日志记录都应通过此类进行。

Member Type Documentation

[since Visindigo 2.0.0] enum class VILogger::Level

定义了日志记录器的日志级别。

定义了日志记录器的日志级别，包括Trace、Debug、Info、Warn、Error、Fatal。

This enum was introduced or modified in Visindigo 2.0.0.

Property Documentation

[since Visindigo 2.0.0] level : Level

获取日志记录器的日志截止级别。 获取日志记录器的日志截止级别。小于此等级的日志将不会被记录。

This property was introduced in Visindigo 2.0.0.
