Visindigo 最小内核 模块

包括了 Visindigo 最小内核的所有类. Visindigo 最小内核是构建 Visindigo 内核的基础。. More...

This module was introduced in Visindigo 2.0.0.

Classes

VIAbstractObject

是Visindigo框架大部分类型的基类，用于在其中提供通用功能. 用户不应该直接使用VIAbstractObject，而应该使用VIObject或者其子类，除非用户需要将一个 已经继承自QObject的类转换为VIObject。

VIConsole

VIConsole为Visindigo提供初级的控制台快捷功能。

VIJsonConfig

VIJsonConfig定义了对Json配置的操作。

VIJsonDocument

此类在VIJsonConfig的基础上增加了一种允许存在默认值的配置文件的处理方式。

VILogger

VILogger定义了Visindigo的日志记录器。

VILoggerManager

VILoggerManager定义了Visindigo的日志记录器管理器。

VIObject

VIObject是Visindigo框架的基类 VIObject是Visindigo框架的基类，所有的Visindigo框架的类都应该继承自VIObject。 如果用户需要将一个已经继承自QObject的类转换为VIObject，可以使用VIAbstractObject。

VIPackage

VIPackage定义了Visindigo的功能集合。

VIQCoreApplication

对QCoreApplication的加壳，以便进行异常处理。
