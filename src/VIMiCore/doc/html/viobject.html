VIObject Class

VIObject是Visindigo框架的基类 VIObject是Visindigo框架的基类，所有的Visindigo框架的类都应该继承自VIObject。 如果用户需要将一个已经继承自QObject的类转换为VIObject，可以使用VIAbstractObject。. More...
